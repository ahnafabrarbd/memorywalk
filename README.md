# memorywalk

a collective publishing platform for psychography journals — visual-textual records of walks that map the psyche against the geography it moves through.

## what this is

each entry is a walk: a sequence of images and text, captured in place, processed through a low-resolution sensor emulation, and published as a hypertext document. walks are connected through geographic, temporal, and semantic collisions — computed relationships that emerge from the content itself, not from social interaction.

there are no comments, no likes, no follows. the only relational artifact is the collision.

## who it's for

anyone who walks and wants to record the experience as a visual-textual document. the platform is open to all contributors via GitHub authentication.

## how to read

visit the site. browse walks at `/walks`, or explore the nexus map at `/nexus` to see where walks intersect. click into a walk and advance through its pages.

## setup

```bash
npm install
npm run dev
```

the site runs at `http://localhost:4321`.

## how to contribute

### write a walk

go to any walk's final page. follow the "write your own" link. you'll need a GitHub account.

### contribute to the codebase

fork the repo, make changes, open a PR. see [CONTRIBUTING.md](CONTRIBUTING.md) for details.

## project values

- constraint over expressibility
- permanence over convenience
- connection over interaction
- the walk over the platform
